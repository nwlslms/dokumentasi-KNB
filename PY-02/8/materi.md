> Review
- [x] Function
    Sebuah fungsi / tugas terbungkus yang bisa kita panggil
    `f(x): 5x + 2`
- [x] Jenisnya
    - Dengan nilai return
    - Tanpa nilai return
- [x] Parameter
    Variabel / data yang dimasukkan di fungsi, kemudian akan dipakai selama fungsi berjalan
    - Aktual
        Variabel / nilai yang kalian masukkin ke fungsi
    - Formal
        Variabel / nilai yang digunakan selama fungsi berjalan
    - Opsional
        Parameter yang bisa dicantumkan / tidak saat fungsi dipanggil, tapi dia memiliki default
- [x] Variabel
    - Global
    Bisa diakses oleh semua fungsi / inden
    - Lokal
    Hanya bisa dipakai di blok tertentu
> Main Course
- [x] Modul / Library
Sebuah file yang bisa kita panggil yang berisi fungsi-fungsi siap pakai, Ex: NumPy, Pandas, socket, math
    - Cara panggil
        ```python
            import NamaModul
            import NamaModul as Alias
            from NamaModul import *
            from NamaModul import yangMauDipakai
        ```
    - Cara Pakai
        ```python
            # Kalau pakai import NamaModul
            from aritmatika
            # Harus panggil nama module dahulu
            aritmatika.fungsi

            # Kalau from ... import 
            fungsi
        ```
