# 🎓 Projek 3: Penilaian Mahasiswa

## 🧾 Narasi
Sebagai seorang pengajar, kamu ingin membuat program sederhana untuk membantu menghitung nilai akhir mahasiswa agar tidak perlu menghitung secara manual lagi.

## 📌 Deskripsi Soal
Buat program yang:
1. Meminta input:
   - Nilai tugas (30%)
   - Nilai UTS (30%)
   - Nilai UAS (40%)
2. Menghitung nilai akhir
3. Menentukan grade:
   - A (≥85)
   - B (≥70)
   - C (≥60)
   - D (≥50)
   - E (<50)

## 🚧 Batasan
- Nilai antara 0 – 100
- Gunakan if-else

## 💻 Contoh Input
Nilai tugas: 80  
Nilai UTS: 75  
Nilai UAS: 90  

## ✅ Contoh Output
Nilai akhir: 83  
Grade: B
