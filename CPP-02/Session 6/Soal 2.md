# 🎯 Projek 2: Game Tebak Angka

## 🧾 Narasi
Kamu sedang membuat game sederhana untuk teman-temanmu agar mereka bisa bermain sambil belajar coding. Game ini adalah tebak angka, di mana pemain harus menebak angka yang benar.

## 📌 Deskripsi Soal
Buat program yang:
1. Menentukan angka rahasia (misalnya: 7)
2. Meminta user menebak angka
3. Memberi petunjuk:
   - Terlalu kecil
   - Terlalu besar
4. Mengulang sampai tebakan benar
5. Menampilkan jumlah percobaan

## 🚧 Batasan
- Angka: 1 – 10
- Gunakan loop
- Hitung jumlah percobaan

## 💻 Contoh Input
Tebakan: 5  
Tebakan: 9  
Tebakan: 7  

## ✅ Contoh Output
Terlalu kecil!  
Terlalu besar!  
Selamat! Anda benar.  
Jumlah percobaan: 3
