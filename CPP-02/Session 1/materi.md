- [x] Struktur pemrograman
    ```cpp
        #include <iostream> //Untuk memasukkan library ke dalam program

        using namespace std;

        int main () {
            cout << "Hello World" << endl; //Menampilkan teks
            cout << "Hallo Dunia" << endl; //endl = endline
            return 0; //Tanda berakhirnya suatu fungsi
        } // Fungsi yang pertama kali dieksekusi
    ```

- [x] Komponen programan
    - Input: Proses memasukkan data ke komputer.
    - Output: Proses menampilkan data dari komputer.
    - Variabel
        - Bilangan bulat
            `integer`: `int`
            ```cpp
                int, short, long, long long, long long int
            ```
        - Bilangan desimal
            `float`: `double`
            ```cpp
                float, double
            ```
        - Huruf / string
            `huruf`:`string`
            ```cpp
                char, char[], string
            ```
        - Logika
            `boolean`
            ```cpp
                bool
            ```
    - Comment
        - Single comment: `//`
            ```cpp
                // Ini comment 1 baris
            ```
        - Line comment: `/*Text*/`
            ```cpp
                /*
                    Nama: Nawwal Salim
                    Asal: Tangerang Selatan
                */
            ```
    - Syntaxis
        - Penamaan variabel
            1. Ga boleh diawali angka
            2. Ga boleh ada spasi
            3. Spasi dipakai `_`
            4. Camel Text: Semua disambung, kata awalnya lowercase, berikut uppercase
        - semicolon `;`
