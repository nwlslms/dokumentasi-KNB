## 🧩 Soal 2: Menghitung Luas Persegi Panjang

### 📝 Deskripsi  
Buatlah program untuk menghitung luas sebuah persegi panjang. Program menerima panjang dan lebar dari pengguna, lalu menampilkan luasnya.

Gunakan tipe data yang sesuai agar hasil perhitungan dapat berupa desimal.

---

### 📥 Input
- Panjang (float)
- Lebar (float)

---

### 📤 Output
- Luas persegi panjang

---

### ⚠️ Batasan
- 0 < panjang ≤ 1000  
- 0 < lebar ≤ 1000  

---

### 💡 Contoh Input
5.5
3.2

---

### 💡 Contoh Output
Luas: 17.6
