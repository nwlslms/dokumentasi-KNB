# ⭐ Projek 5: Pola Bintang

## 🧾 Narasi
Untuk melatih logika looping, kamu ingin membuat program yang bisa menggambar pola bintang. Program ini bisa digunakan untuk latihan dasar pemrograman.

## 📌 Deskripsi Soal
Buat program yang:
1. Meminta jumlah baris
2. Menampilkan pola segitiga bintang

## 🚧 Batasan
- Jumlah baris: 1 – 20
- Gunakan nested loop

## 💻 Contoh Input
Masukkan jumlah baris: 4
